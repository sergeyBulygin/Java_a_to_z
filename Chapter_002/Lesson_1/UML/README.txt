В пакете ru.sbulygin.models реализованны следующие классы:
1. Item -    базовый класс описывающий модель заявки.
2. Tracker - класс используеться как хранилище заявок и содержит методы работы с ними.
3. Task -    класс содержащий заявки принятые для исполнения.
4. Bug -     класс исправляющий ошибки в заявках.
5. CheckInput - класс принимающий введеные данные пользователем и проверяющий корректность введенных данных.

В пакете ru.sbulygin.start реализован класс StartUI в котором реализованно консольное меню программы.
